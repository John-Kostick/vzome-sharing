This is a study of the geometry of pyramids.  Specifically, pyramids that resemble the [Great Pyramid of Giza](https://en.wikipedia.org/wiki/Great_Pyramid_of_Giza).  While the original intended proportions can only be surmised, there is a consensus that it can be described as a rirht pyramid with base width 2 inits and altitude √𝜑 units. Using this proportion, it follows that a line from the center of a base edge to the apex is 𝜑 units, and the slpoe is Arc Cos 1/𝜑, 51.827...°.

Below is a straight-edge-compass construction for a face of a pyramid.  A 1,2,√5 triangle is constructed, and it's hypoteneuse (green) is rotated to add to the short side to define Arc Tan 2/(√5+1) = 31.717...° as half angle of the apex, 63.43...°, the same angle as in a golden rhombus.
